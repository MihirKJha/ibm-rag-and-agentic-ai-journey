# About This Repository

This repository documents my journey through the **IBM RAG & Agentic AI Professional Certificate**.

After completing the **IBM AI Engineering Professional Certificate**, this repository focuses on applying those fundamentals to build modern AI applications using **Large Language Models (LLMs)**, **LangChain**, **Retrieval-Augmented Generation (RAG)**, **AI Agents**, and **Agentic AI**.

Rather than simply completing another certification, my goal is to understand how enterprise AI systems are designed, developed, and deployed using modern AI frameworks and production-ready engineering practices.

This repository combines:

- 📓 Hands-on notebooks
- 💻 Practical implementation projects
- 📚 Curated technical notes
- 🏗️ AI application architecture
- 🤖 Enterprise AI engineering concepts
