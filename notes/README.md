# AI Notes

> A curated collection of technical notes created while completing the **IBM RAG & Agentic AI Professional Certificate**. These notes consolidate key concepts, architecture patterns, and engineering best practices to build a reusable knowledge base for Enterprise AI Engineering.

---

# 📌 About

Unlike the course modules, which focus on **hands-on notebooks and implementation projects**, this directory contains **concept-oriented notes** organized by technical domains.

Rather than duplicating content across modules, related concepts are consolidated into a single note, making them easier to maintain, extend, and reference as the repository grows.

The objective of these notes is to:

- Build a structured AI engineering knowledge base.
- Consolidate concepts from multiple learning resources.
- Provide quick revision material for interviews and certifications.
- Document architecture patterns and best practices.
- Serve as a practical reference while building Enterprise AI applications.
